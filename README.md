🚗 Vehicle Rental System API

A full-stack backend API for a Vehicle Rental System built with Node.js, Express, TypeScript, PostgreSQL (Neon).
It supports user management, vehicle management, and booking system with automatic return handling.

🚀 Features
👤 User registration & management
🚗 Vehicle management (CRUD)
📦 Booking system (rent vehicles)
🔄 Auto-mark returned bookings
📊 PostgreSQL database (Neon cloud)
⚡ RESTful API architecture
🧠 TypeScript type safety
🔐 Environment-based configuration
